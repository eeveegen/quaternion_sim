This program piece consists of 2 files: an internal script run by Blender, and an external interface that can more easily access all of the computer's resources, including serial.

In order to get a good performance, the internal script should be ran within blender's context, so the file's contents here are just for show. 
To set it up: 
- Open Blender project 
- If not done: Edit -> Preferences -> Interface -> Display -> Enable "Developer Extras" and "Python Tooltips"
- Open Text Editor within Blender (TBA how)
- Copy blender_int_ctrl.py's contents within the text editor
- Start internal and external file shortly after one another (Blender internal file started by "play" arrow above Text Editor)
